# SHIRO Technologies - Next.js Application

## Quick Start

```bash
npm install
npx shadcn-ui@latest init
npx shadcn-ui@latest add button card input select
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- ✅ Next.js 14 App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Cookie Consent (GDPR compliant)
- ✅ Google Analytics (G-1LL63ZEXDY)
- ✅ Homepage with QA Calculator
- ✅ Products Page (7 products)
- ✅ AI Transformation Page with Value vs Viability Tool
- ✅ Responsive Design
